Why A Glossary Exists

Enterprise finance collapses different meanings into the same ISO symbol unless states are named.

Two postings may both display USD while representing quoted exposure, transaction denomination, settlement cash movement, account custody, ledger maintenance, reporting presentation, group consolidation, or terminal audit defensibility. The code repeats; the institutional meaning does not.

AccountCcy.com maps those meanings through the Currency State Control Framework and the CCY State Chain. This hub defines the anchor terms so navigation, risk discussion, and CFO governance share one disciplined reference layer.

Individual deep-dive term pages will follow as the corpus expands. This page is the authoritative orientation surface — not a flat dictionary, but the terminology constitution for the asset.

Architecture

CCY State Chain

Eight ordered institutional states from quoted market signal through audit reality; each state asks a different custody question and admits different failure modes if blurred with its neighbors.

Methodology

Currency State Control Framework

The original AccountCcy structure for naming currency states, transitions, and control obligations — treating account currency as a discipline, not a passive configuration field.

Lens

Currency State

The institutional role currency occupies at a point in the finance chain (quotation, obligation, settlement, account, ledger, reporting, consolidation, audit) — distinct from the mere ISO code printed beside an amount.

State 01

Quoted Currency

The currency in which a price or rate is expressed before a transaction creates obligation — market signal prior to accounting custody.

State 02

Transaction Currency

The currency that denominates the financial event — where obligation takes formal identity inside commerce or operations before full ledger settlement narratives conclude.

State 03

Settlement Currency

The currency of actual cash or settlement rail movement — divergence from transaction currency seeds FX consequences that propagate into postings.

State 04 · Control

Account Currency

The currency assigned to an account for recording, reconciling, and controlling balances — the custody gate where external movement enters internal accounting discipline.

State 05

Ledger Currency

The currency in which books and balances are maintained at entity level — frequently aligned with functional currency policy but not interchangeable with account or reporting layers.

Standards Context

Functional Currency

The currency of the primary economic environment in which an entity operates — an accounting determination that interacts with ledger policy yet remains analytically distinct from every account-level assignment.

State 06

Reporting Currency

The currency of financial presentation for management, regulatory, investor, or internal reporting — translation and policy choices surface here.

State 07

Consolidation Currency

The group currency into which entity reporting layers unify — maximum translation complexity and elimination logic converge at this altitude.

State 08

Audit Reality

Terminal custody state: figures trace to evidence, reconciliations close defensibly, documentation survives independent challenge — the outcome weak state discipline silently forfeits.

Lexicon Control
